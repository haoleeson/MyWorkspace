# 1. 机器学习是什么
- 机器从数据中学习
- 机器学习（≈ 寻找一个函数）

f( 一段语音 ) = 语义

f( 一张图片 ) = 识别对象

f( 棋局 ) = 最优解

f( 语义 ) = 反馈语义

# 2. 机器学习的三种分类
- **监督学习（有标签）**

任务驱动：识别、分类、回归

- **非监督学习（无标签）**

数据驱动：聚类

- **强化学习**

与环境互动

# 3. 理解机器学习的框架
以图像识别为例，讲解监督学习框架的四个步骤：

- 收集数据
- 选择模型。模型：函数的集合（无穷个）
- 调整参数。定义模型中函数的好坏
- 找到最优（函数 f*）
